项目名为微型博客
目前实现功能
1，用户注册及登录
2，密码通过hash加密传入数据库
3，用户编辑个人信息，上传头像（简单处理，并未完善）
4，用户可编辑属于自己的内容，目前仅支持插入，没有删除和更新内容的功能

环境
项目基于 Python2.7，sqlite3，其余依赖请参考requirements.txt

错误
如运行过程中出现错误，请自行解决
